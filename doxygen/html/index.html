API Reference
API Summary

The basic WPS API sequence can be summarized to the following calls:

  • WPS_load()
  • WPS_set_key()
  • WPS_location()
  • WPS_unload()

Periodic and IP location:

  • WPS_periodic_location()
  • WPS_ip_location()

Configuration:

  • WPS_set_tiling()
  • WPS_set_tunable()

Geofencing:

  • WPS_geofence_set()
  • WPS_geofence_cancel()
  • WPS_geofence_cancel_all()

Offline location:

  • WPS_offline_token()
  • WPS_offline_location()
